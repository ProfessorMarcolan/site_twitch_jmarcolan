Uma linguagem de programação é caracterizado por uma sitaxe e pelo seu dicionário. Então, o javascript vai ter uma série de palavras reservadas (palavra function ) que tem um signifcado construido (que significa função) e uma forma de juntar essas palavras em uma frase. O looping principal.

O que significa que o javascript possui um looping principal no qual você não QUER de nenhum jeito fique parado exectuando uma unica tarefa. Por que isso significa o apocalipse. Basicamente todo o Navegador para de funcionar até o momento que você deleta a atividade . O que é o looping principal não blocante, significa que não tem tarefas que bloqueiam o código.

Fracamente e dinamicamente tipada

Isso signifca que para a criação de variaveis não se precisa declarar espeficamente qual vai ser o tipo de várivel. Por exemplo, você não precisa explificamente disser para o javascript que você gostaria que a variavel fosse um texto. Ele consegue, em tempo de execução determinar que a aquela variavel é do tipo texto. E ele á fracamente tipada porque após ela ser criada tanto faz o tipo . Referencias (https://pt.wikipedia.org/wiki/Linguagem_tipada, )

tipos de variaives no javascript

O javascript possui 3 tipos de variaveis primiticas que são atribuidas em tempo de execução, booleana (bool), numérica (-number) e de texto (string). Existe algumas formas de constuir uma variavel por esse tutorial iremos entender usando a palavra reservada let (tem const e var que iremos discutir em outros tutoriais) Exmeplo let variavel_boleana = true //variavel booleana let variavel_string = "oi mundo" //variavel string let variavel_numero = 123123 let variavel_numero2 = 132.231 Esse tipos primitivos são os tijolos para construir estruturas de dados. Geralmente os programas usam estrutuas de dados. Todas as estruturas de dados do javascript são typos de objetos diferentes . Eles se dividem em listas de objetos, dicionarios e funções. Cada um desses tópicos veremos nas proximos tutoriais.

O tortoiseGit é um controlador de versão baseado no TortoiseSVN para Windows. Ele é um programa de código aberto e pode ser usado no versionamento de qualquer tipo de documento pois suas funções não estão atreladas a nenhum IDE especifico mas sim ao menu de contexto do Windows Explorer .

Materiais necessário para o acompanhamento do Tutorial

Para esse tutorial será necessário:

  • Um computador com o sistema operacional Windows 7.
  • Ter instalado o git-for-windows .
  • O download do software turtoiseGit .

Tutorial de instalação do TortoiseGit

  • Passo 1: Clique no arquivo TortoiseGit te o execute;
  • Passo 2: Na janela de inicio, figura 1a, clique no botão “next” para começar a configuração da instalação do TorotiseGit;
  • Passo 3: Na janela de informação sobre a licença, figura 1b, clique no botão “next>” para aceitar a licença;
  • Passo 4: Na janela de customização, figura 2a, clique no botão “browse” para escolher a localização da instalação do TortoiseGit, além disso, confira as principais caracteristicas que serão instaladas e as suas dependências. Então, clique no botão “next” para continua a configuração da instalação.
  • Passo 5: Na janela para o inicio da instalação do TortoiseGit, figura 2b, clique no botão “Install” para começar a instalação”.
  • Passo 6: Na janela de instalação, figura 3a, e espera a instalação do TortoiseGit completar. Então, clique no botão “next”.
  • Passo 7: Na janela fim da instalação, figura 3b, clique no botão Finish para fechar a janela de instalação.

Passo 10: Clique com o botão direito dentro de alguma pasta e verifique se no menu de contexto do windows Explorer está habilitado as funções: “Git clone”,” Git Create repository here”, “TortoiseGit”.

Pronto, está instalado o TortoiseGit em seu computador.
